    \section{Problem 1}\label{problem-1}

\subsection{\texorpdfstring{\textbf{Problem
Statement:}}{Problem Statement:}}\label{problem-statement}

Let \(A\) be an \(m \times d\) matrix, and let \(X = AA^T\). Assume that
\(X\) has \(d\) distinct, non-zero eigenvalues. Assume that \(m ≫ d\).
In order to find the eigendecomposition of \(X\), we will need to find
the eigendecomposition of an \(m \times m\) matrix. Since \(m\) is much
larger than \(d\), this is slow. Give an algorithm for finding the
eigenvectors and eigenvalues of \(X\) that only requires computing the
eigendecomposition of a \(d \times d\) matrix. You can use simple matrix
operations and assume that you have an eigendecomposition ``black box''
subroutine, but avoid using the SVD as a black box.

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsection{\texorpdfstring{\textbf{Solution}}{Solution}}\label{solution}

\subsubsection{Intuition}\label{intuition}

Since the rank of \(X\) is at most \(d\) (because \(A\) only has \(d\)
columns), we only need to compute the eigendecomposition of a smaller
\(d \times d\). This is because while \(AA^T\) is size \(m \times m\),
\(A^TA\) is size \(d \times d\), and the two matrices share the same
eigenvalues because they are both derived from \(A\). Thus, we can use
the calculated eigenvalues for \(B = A^TA\) to construct the
eigenvectors of \(X\).

\subsubsection{Algorithm}\label{algorithm}

\paragraph{\texorpdfstring{\textbf{Compute
B}}{Compute B}}\label{compute-b}

Given \(A\), an \(m \times d\) matrix, a \(d \times d\) matrix \(B\)
will be computed as:

\[ B = A^TA\]

\paragraph{\texorpdfstring{\textbf{Compute Eigendecomposition of
B}}{Compute Eigendecomposition of B}}\label{compute-eigendecomposition-of-b}

The eigendecomposition of \(B\) will be computed as:

\[B = QDQ^T \]

Where \(D\) is a diagonal matrix containing the eigenvalues of \(B\),
and \(Q\) is an orthogonal matrix whose columns are the eigenvectors of
\(B\).

\paragraph{\texorpdfstring{\textbf{Derive the Eigenvectors and
Eigenvalues of X from the Eigendecomposition of
B}}{Derive the Eigenvectors and Eigenvalues of X from the Eigendecomposition of B}}\label{derive-the-eigenvectors-and-eigenvalues-of-x-from-the-eigendecomposition-of-b}

\textbf{1. Define an eigenvector of \(B\)}

Let's fix \(\mathbf{v}_i\) which is an eigenvector of \(B\) in
\(d\)-dimsional space corresponding to the eigenvalue \(\lambda_i\),
using the eigenvector and eigenvalue definition
\(B\mathbf{v}_i =\lambda_i \mathbf{v}_i\):

\[B\mathbf{v}_i = A^TA\mathbf{v}_i = \lambda_i \mathbf{v}_i \]

\textbf{2. Define a new vector \(\mathbf{u}_i\) to map \(\mathbf{v}_i\)
into higher dimensional space}

Let's define a vector \(\mathbf{u}_i\) in the larger \(m\)-dimensional
space as:

\[ \mathbf{u}_i = A\mathbf{v}_i \]

Where multuplying \(\mathbf{u}_i\) by \(A\) maps the vector
\(\mathbf{v}_i\) from \(d\)-dimensional space to the \(m\)-dimensional
space.

\textbf{3. Show that \(\mathbf{u}_i\) is an eigenvector of \(X\)}

To prove that \(\mathbf{u}_i\) is an eigenvector of \(X\), we must show:

\[ X\mathbf{u}_i = \lambda_i\mathbf{u}_i\]

Substituting \$ \mathbf{u}\_i = A\mathbf{v}\_i \$ and \$ X = AA\^{}T \$
into the equation:

\[ AA^T(A\mathbf{v}_i) = \lambda_i(A\mathbf{v}_i) \]

Since \(B = A^TA\):

\[ A(B\mathbf{v}_i) = \lambda_i(A\mathbf{v}_i) \]

And since \(B\mathbf{v}_i = \lambda_i\mathbf{v}_i\):

\[ A (\lambda_i\mathbf{v}_i) = \lambda_i(A\mathbf{v}_i) \]

Which is the same as:

\[ \lambda_i (A\mathbf{v}_i) = \lambda_i(A\mathbf{v}_i) \]

Proving that:

\[ X\mathbf{u}_i = \lambda_i\mathbf{u}_i\]

    \section{Problem 2}\label{problem-2}

\subsection{\texorpdfstring{\textbf{Problem
Statement:}}{Problem Statement:}}\label{problem-statement}

In this problem we explore some relationships between SVD, PCA and
linear regression.

\begin{enumerate}
\def\labelenumi{(\alph{enumi})}
\item
  True or false: linear regression is primarily a technique of
  supervised learning, i.e.~where we are trying to fit a function to
  labeled data.
\item
  True or false: PCA is primarily a technique of unsupervised learning,
  i.e.~where we are trying to find structure in unlabeled data.
\item
  True or false: SVD is primarily an operation on a dataset whereas PCA
  is primarily an operation on a matrix.
\item
  A common problem in linear regression is multicollinearity, where the
  input variables are themselves linearly dependent. For example,
  imagine a healthcare data set where height is measured both in inches
  and centimetres. This is a problem because there may now be multiple
  \(w\) satisfying \(y = w · x\). Explain how you could use a
  preprocessing step to solve this problem.
\end{enumerate}

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsection{\texorpdfstring{\textbf{Solution}}{Solution}}\label{solution}

\subsubsection{Part A}\label{part-a}

\textbf{True}.

Linear regression is a supervised learning technique, since we have a
training set \(\mathbf{S}\) which has data points and associated labels.
The goal is to find the best fitting line to the labeled training set so
we can use the model to predict on unseen data.

\subsubsection{Part B}\label{part-b}

\textbf{True}.

Since PCA is a technique to find patterns and information in a matrix of
unlabeled data, it is a technique of unsupervised learning. The goal is
to find the directions upon which the data has the most variance
(information).

\subsubsection{Part C}\label{part-c}

\textbf{False}.

SVD and PCA are both matrix operations. SVD is a technique for
decomposing a matrix into a simpler structure (and also performing the
eigendecomposition at the same time!).

\subsubsection{Part D}\label{part-d}

To address the multicollinearity issue, PCA can be performed as a
preprocessing step. The PCA will find the directions for which the data
has the most variance, or provides the most information about the
dataset, and reduces the redundancy of the dataset in the process.
Furthermore, \textbf{the resulting principal components will be linearly
independent of one another, eliminating the multicollinearity}.
